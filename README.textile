rdb-tools is a parser for Redis *.rdb files. 

The API is similar to a XML SAX parser. You create a callback to receive events, 
and then pass your callback to the parser. 

h2. Sample Code
<pre><code>
from rdbtools import RdbParser, RdbCallback

class PrintKeys(RdbCallback) :
    def set(self, key, value, expiry):
        print(key)

    def start_hash(self, key, length, expiry):
        print(key)

    def start_set(self, key, cardinality, expiry):
        print(key)

    def start_list(self, key, length, expiry):
        print(key)

    def start_sorted_set(self, key, length, expiry):
        print(key)

callback = PrintKeys()
parser = RdbParser(callback)
parser.parse('/var/redis/6379/dump.rdb')
</code></pre>
